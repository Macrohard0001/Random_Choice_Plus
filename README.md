# **Random_Choice_Plus**
###  **随机点名plus版**

![image](https://github.com/user-attachments/assets/9441861e-0761-40be-93e7-a53a8b0b6509)

###**软件亮点：**

* 为教学而生，适配教师需求

* 优良的点名性能

* 人性化交互界面，美观而又方便

* 自定义程度高，打造属于你们班的个性化点名器

* 智能逻辑，单次使用不重复点名（除非你重置名单）
